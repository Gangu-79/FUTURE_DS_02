FUTURE_DS_02 — Customer Retention & Churn Analysis Dashboard

Project Overview

This project focuses on analyzing customer churn and retention patterns for a subscription-based business using Power BI.

The dashboard helps identify:

- Customer churn trends
- Retention behavior
- High-risk customer groups
- Contract-based churn analysis
- Payment method impact
- Internet service analysis

Tools & Technologies Used

- Power BI
- DAX
- Data Visualization
- Data Cleaning
- Business Intelligence

Key Features

- Interactive Dashboard
- KPI Metrics
- Churn Analysis
- Retention Insights
- Contract Type Analysis
- Payment Method Analysis
- Internet Service Analysis
- Dynamic Filters & Slicers

Dashboard KPIs

- Total Customers
- Churned Customers
- Active Customers
- Churn Rate
- Retention Rate

Key Insights

- Month-to-month contract customers showed higher churn rates.
- Electronic check users had higher churn behavior.
- Long-term contract customers demonstrated better retention.
- Internet service type influenced customer churn patterns.

Project Outcome

Developed an interactive Power BI dashboard to generate business insights and support customer retention strategies.
